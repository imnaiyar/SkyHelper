SkyHelper is a discord bot designed to provide (useful?) information about the game [Sky: Children of the Light](https://thatskygame.com)

### Features

- Shards
- In-game event times
- Guides
- Seasonal currency calculator
- Shards Calendar
- Live Shard/In-game event times updates
- Reminders (Grandma, Geyser, etc...)
- Supports Multiple Languages
