SkyHelper is a feature-rich Discord bot designed to enhance your Sky: Children of the Light experience. This documentation will guide you through everything you need to know to get started and utilize the bot's functionalities.

## Features

SkyHelper offers a variety of helpful features to streamline your Sky adventures:

- **Shards**: Track and manage your in-game Shards
- **Shards Calendar**: Stay updated on upcoming Shards events
- **Seasonal Currency Calculator**: Optimize your seasonal currency earnings
- **In-game Events**: Get real-time information on event times and countdowns
- **Reminders**: Set reminders for Geysers, Grandma visits, and other important in-game activities (requires additional setup)
- **Live Updates**: Receive notifications for critical in-game updates
- **Multilingual Support**: Access features and information in multiple languages
- **And Much More**: SkyHelper is constantly being developed with new features in the pipeline

## Getting Started

1. **Invite the bot** to your Discord server using our [invitation link](https://skyhelper.xyz/invite)
2. **Explore commands** by typing `/help` to see all available commands
3. **Set up features** like reminders and live updates if you have the necessary permissions
4. **Enjoy** enhanced Sky: Children of the Light experience!
