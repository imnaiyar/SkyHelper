SkyHelper is a Discord bot designed to provide useful information about the game [Sky: Children of the Light](https://thatskygame.com). The bot aims to enhance your gameplay experience by providing real-time information, helpful tools, and community features.

## Features

- **Shards**: Track shard locations and timings
- **In-game Event Times**: Get accurate timings for various events
- **Guides**: Access comprehensive guides for different aspects of the game
- **Seasonal Currency Calculator**: Plan your seasonal spending efficiently
- **Shards Calendar**: Never miss a shard event again
- **Live Updates**: Receive real-time updates for shards and in-game events
- **Reminders**: Set up personalized reminders for Grandma, Geyser, and other events
- **Multilingual Support**: Available in multiple languages to serve the global Sky community

## Development

SkyHelper is actively maintained and regularly updated with new features and improvements. The bot is built with modern technologies and follows best practices for Discord bot development.

### Technology Stack

- **Language**: TypeScript/JavaScript
- **Framework**: Discord.js
- **Database**: MongoDB
- **Hosting**: Self-hosted with Docker
