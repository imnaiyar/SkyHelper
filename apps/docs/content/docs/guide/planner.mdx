# Sky Game Planner

An interactive, comprehensive planner for browsing and exploring all Sky: Children of the Light game content in one place.

## Overview

The Sky Game Planner is a powerful feature that lets you:

- Browse all spirits, items, events, and shops
- Filter and search content interactively
- Navigate through game areas and realms
- Plan your Sky journey efficiently
- Access detailed information quickly

<Callout type="info">
  **All-in-One Tool**: The planner consolidates spirits, cosmetics, events, seasons, shops, and more into a single,
  easy-to-navigate interface.
</Callout>

## Command

### `/planner` - Interactive Game Planner

Access the interactive planner with search and navigation features.

**Subcommands**:

#### `search` - Search for anything

```
/planner search query:Dancing Performer
```

Search for spirits, items, events, shops, or navigate to specific sections.

## State Persistence

The planner maintains state across interactions:

- **Filter Settings**: Remain active during navigation
- **Pagination**: Remembers your page position
- **Navigation History**: Full back-button support
- **User Isolation**: Your planner state is independent

<Callout type="info">
  **Session Based**: Each user has their own planner session. Your navigation doesn't affect others using the planner in the same
  channel.
</Callout>

## Performance & Limits

The planner is optimized for:

- **Fast Loading**: Quick response times
- **Large Datasets**: Handles all game content
- **Smooth Filtering**: Instant filter updates
- **Pagination**: Manages large result sets

<Callout type="warning">
  **Discord Limits**: Due to Discord's interaction limits, very long browsing sessions may time out. Simply run the command again
  to continue.
</Callout>
