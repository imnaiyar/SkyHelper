# SkyHelper Feature Guide

Welcome! This guide explains all the features SkyHelper offers to enhance your Sky: Children of the Light experience.

## What is SkyHelper?

SkyHelper is your companion for Sky: Children of the Light. It helps you:

- Track shards and in-game events
- Find spirits and plan unlocks
- Calculate seasonal progress
- Set up reminders and live updates
- Play games and have fun
- And much more!

<Callout type="info">
  **New to SkyHelper?** Start with [Shards](/guide/shards) and [Events](/guide/events) to understand the core features!
</Callout>

## Getting Started

### First Steps

1. Pick a feature that interests you
2. Read its guide to understand what it does
3. Try it out in Discord
4. Explore related features

### Feature Combinations

Features work great together:

- **Shards + Reminders**: Track shards and get notified when they appear
- **Events + Live Updates**: See schedules and get auto-updating displays
- **Spirits + Planner**: Find spirits and explore their complete trees
- **Seasonal + Events**: Plan seasons and track daily quest guides

### Learning Path

Suggested order for new users:

1. **Shards** - Learn the basics of tracking
2. **Events** - Understand all Sky events
3. **Spirits** - Discover how to find and unlock spirits
4. **Utilities** - Get helpful tools
5. **Advanced Features** - Explore planner, reminders, etc.

## Important Notes

### Server vs User Features

Most features work anywhere, but some require servers:

- **Work Everywhere**: Shards, Events, Spirits, Seasonal, Planner, Games, Utilities, Language
- **Server Only**: Live Updates, Reminders (need Manage Server permission)

<Callout type="warning">
  Server features require the bot to be added to the server and you need Manage Server permissions to set them up.
</Callout>

### Language Support

SkyHelper works in multiple languages:

- Set your personal language preference
- Server admins can set server defaults
- All features translate to your chosen language
- Learn more in [Language Support](/guide/language)
